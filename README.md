# 归梦工作室通用后台管理系统

本项目是使用 [Ant Design Pro](https://pro.ant.design) 初始化的。以下是使用的快速指南。

## 环境准备

安装 `node_modules`：

```bash
npm install
```

或

```bash
yarn
```

## 提供的脚本

Ant Design Pro 提供了一些有用的脚本，帮助您快速启动和构建 Web 项目、代码风格检查和测试。

脚本在 `package.json` 中提供。可以安全地修改或添加其他脚本：

### 启动项目

```bash
npm start
```

### 构建项目

```bash
npm run build
```

### 检查代码风格

```bash
npm run lint
```

您还可以使用脚本自动修复一些 lint 错误：

```bash
npm run lint:fix
```

### 测试代码

```bash
npm test
```
